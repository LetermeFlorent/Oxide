Your Workspace,

Oxide is a high-performance developer environment bridging the gap between web flexibility and native power. Built with Tauri v2 and React 19.

Get Started Free View Source

Everything you need to build.

Designed for developers who care about performance and aesthetics.

Native PTY Terminal

Real Bash/Shell execution via Rust backend. Stream output directly to Xterm.js with zero latency.

Multi-Project

Open and manage multiple projects simultaneously. Your workspace state is automatically persisted.

Smart Preview

High-performance rendering for PDF, Markdown with live preview, and Monaco code editor.

Native power,
Web flexibility.

Oxide leverages the best of both worlds. The core is written in **Rust** for safety and speed, while the UI is powered by **React 19** for a smooth user experience.

Start building now.

Get Oxide up and running in less than 2 minutes.

1

Clone: git clone https://github.com/LetermeFlorent/Oxide.git

2

Install: Navigate to /app and run npm install

3

Launch: Run npm run tauri dev
